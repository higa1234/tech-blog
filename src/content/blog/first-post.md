# Astroで技術ブログを作り始めた

Astroを使って、個人の技術ブログを作り始めました。

このブログでは、学んだことや調べたことを、あとから見返せる形で残していきます。

## 書いていきたいこと

主に以下のような内容を書いていく予定です。

- PostgreSQL
- Web開発
- WSL
- Git / GitHub
- 開発中に詰まったこと

## なぜブログを作るのか

調べたことをその場限りで終わらせず、あとから使える知識として整理するためです。

また、同じことで困っている人の助けになる記事も書けたらよいと思っています。
